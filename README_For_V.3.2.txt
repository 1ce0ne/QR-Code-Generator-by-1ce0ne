Описание приложения

Генератор QR Кодов - это удобное приложение, позволяющее создавать QR-коды в формате PDF для заданного диапазона номеров.
Каждый QR-код будет содержать ссылку, которая будет включать номер самоката в специальном формате.
Приложение имеет интуитивно понятный интерфейс и позволяет настраивать размеры QR-кодов,
а также выбирать папку для сохранения результатов.

Как использовать приложение

1. Запуск приложения

- Запустите файл приложения (QR_Code_Generator.exe).

2. Настройка параметров QR-кодов

- Выбор размера квадрата:
    - В главном окне приложения вы увидите два флажка "Квадрат 2.0 см" и "Квадрат 2.6 см"
    При нажатии на одну из них, применятся настройки форматирования будущего QR кода.

3. Установка диапазона номеров

- Начальный и конечный номера:
    - Введите начальный и конечный номера для генерации QR-кодов в соответствующие текстовые поля.
    Убедитесь, что начальный номер меньше или равен конечному номеру.

4. Установка Кол-ва столбцов и строк в Вашем PDF файле

- Выбор кол-ва столбцов в QR коде:
	- Изменив положение ползунка, выберите нужное Вам кол-во столбцов из возможных для данного размера QR кодов

- Выбор кол-ва строк в QR коде:
 	- Изменив положение ползунка, выберите нужное Вам кол-во строк из возможных для данного размера QR кодов

5. Выбор папки для сохранения

- Папка для сохранения:
    - Нажмите кнопку "Выбрать папку" для выбора папки, в которой будет сохранен сгенерированный PDF файл.
    После выбора пути, он отобразится в соответствующем текстовом поле.

6. Название PDF файла

- Введите желаемое имя для вашего PDF файла. Убедитесь, что имя не пустое.

7. Генерация PDF файла

- После заполнения всех полей, нажмите кнопку "Сгенерировать PDF файл".
Приложение проверит введенные данные и создаст PDF файл с QR-кодами в заданной папке.

8. Успешное завершение

- После успешной генерации QR-кодов появится сообщение об успешном завершении операции.
Вы сможете найти сгенерированный PDF файл по указанному вами пути.


Примечания

- Убедитесь, что у вас достаточно прав для записи в выбранную папку.
- В случае ошибок (например, если введены некорректные числа или оставлены пустые поля),
приложение выдаст соответствующее сообщение об ошибке.
- Временные файлы приложения сохраняются в папке с exe файлом. Рекомендуется перед каждым использованием вручную очищать данную папку.
